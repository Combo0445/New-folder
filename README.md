# 🐾 IT 08-1: Pet Feed Project

ระบบ Tesecase ที่พัฒนาด้วย **Go (Fiber)** และ **Angular** เชื่อมต่อฐานข้อมูล **SQL Server**
## 🛠️ Tech Stack
### **Backend (API)**
- **Language:** Go (Golang)
- **Framework:** Fiber v2
- **Database:** Microsoft SQL Server
- **ORM:** GORM

### **Frontend (UI)**
- **Framework:** Angular 17+
- **Features:** Standalone Components
